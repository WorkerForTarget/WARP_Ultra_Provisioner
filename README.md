## 📌 О проекте

**WARP Ultra-Fast Provisioner** — это десктопное приложение с удобным графическим интерфейсом, которое автоматически регистрирует устройства в сети Cloudflare WARP и генерирует готовые WireGuard-конфиги.

Больше никаких ручных запросов через curl, никакого копирования ключей вручную. Просто указал количество — получил рабочие `.conf` файлы.

### Возможности

- ✅ Массовая генерация WARP+ конфигов (от 1 до 100 штук за раз)
- ✅ Автоматическая генерация ключей Curve25519
- ✅ Регистрация устройств через официальный API Cloudflare
- ✅ Сохранение в любую выбранную папку
- ✅ Индикатор прогресса и подробное логирование
- ✅ Тёмная тема
- ✅ **Готовый .exe файл** — не требует установки Python

## ⚠️ ВАЖНО: О блокировке Cloudflare

**Если вы находитесь в стране, где Cloudflare WARP заблокирован (Россия, Иран, Китай и др.), программа не сможет напрямую зарегистрировать конфиги.**

### 🔍 Как понять, что у вас блокировка?

При запуске программы вы увидите в логах:

!!! ВНИМАНИЕ: Сервер регистрации недоступен !!!

!!! ВКЛЮЧИТЕ РАБОЧИЙ VPN, ИНАЧЕ КОНФИГИ БУДУТ '0 БАЙТ' !!!


Конфиги могут создаваться, но они будут **нерабочими** (в интерфейсе WireGuard будет "Передача: 0 байт").

### 🛡️ Что делать?

### Способ: Включить VPN перед запуском (рекомендуется)

1. **Включи любой VPN** (например, бесплатные: TunnelBear, ProtonVPN, PlanetaVPN)
2. **Убедись, что выбрана страна, где WARP не заблокирован** (Германия, Нидерланды, США)
3. **Запусти программу** через VPN
4. Конфиги сгенерируются рабочие, потому что запросы будут идти через VPN-канал

**Почему это работает:** Подключение к API Cloudflare идёт через сервер VPN, который находится за пределами блокировки.

## 🚀 Быстрый старт

### Вариант A: Используйте готовый `.exe` файл (для Windows)

1. Скачайте последний релиз из раздела **Releases**
2. Запустите `Warp_Provisioner.exe`
3. **ВАЖНО:** Если Cloudflare заблокирован в вашей стране — сначала включите VPN
4. Укажите количество профилей и папку для сохранения
